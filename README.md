Coin Collection Game

Overview

This is a simple coin collection game developed in Unity. The game features an interactive coin collection system where players can collect coins, track their progress via a UI, and advance to the next level once all coins are collected.



Features

Custom-made Game Objects: All game objects, including coins, UI elements, and other assets, were created from scratch before the game development process began.

Coin Collection System: Coins are tracked, and the UI updates in real-time as the player collects them.

UI Management: The coin count is displayed using TextMeshPro and updates dynamically.

Scene Progression: The game loads the next scene automatically when all coins are collected.

Smooth Animations and Effects: Optional enhancements include sound effects and particle animations upon coin collection.




Development Process

All assets used in this game, including 3D models, UI elements, and scripts, were entirely created from scratch before the actual coding of the game started. This ensures originality and uniqueness in the game design.






How to Play

Move your player character towards a coin.

When touching a coin, it will be collected, and the UI counter will update.

Once all coins are collected, the game automatically moves to the next level.
